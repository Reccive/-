# 高压线巡检机器人设计与仿真分析平台

> 基于《轮抱式高压线巡检机器人的机构设计与研究》的集成化设计与仿真分析工具

## 📖 项目简介

本项目是一个用于高压线巡检机器人的设计与仿真分析的集成化平台，采用Python + PyQt5开发，提供参数化设计、运动仿真、样机控制和知识管理四大核心功能模块。

**开发依据**：《轮抱式高压线巡检机器人的机构设计与研究》

**版本**：v1.0.0
**发布日期**：2026-03-25

## ✨ 主要功能

### 1. 参数化设计与计算
- 抱紧机构参数计算（推程、抱紧力、强度校核）
- 驱动电机选型计算（水平、爬坡）
- 关键部件强度分析（连杆应力分布）
- 可视化参数曲线与曲面图

### 2. 运动仿真与分析
- 爬坡稳定性仿真（不同倾角和线径下的稳定性分析）
- 越障过程仿真（三臂轮流式越障运动）
- 运动学/动力学参数可视化
- 性能指标对比分析

### 3. 样机测试与控制
- 串口通信控制（物理样机）
- 虚拟模式仿真（无硬件时）
- 实时数据采集与日志记录
- 远程控制指令执行

### 4. 知识库与方案管理
- 设计指标数据库（国内外研究综述）
- 方案保存与加载（JSON格式）
- 实验记录管理
- 历史方案对比分析

## 🛠️ 系统要求

### 硬件要求
- **CPU**：Intel i5 或同等处理器
- **内存**：8GB RAM（推荐16GB）
- **硬盘空间**：2GB可用空间
- **显示器**：分辨率 1920×1080 或更高
- **可选**：串口转USB适配器（用于连接硬件样机）

### 软件要求
- **操作系统**：Windows 10/11 64位
- **Python环境**：Python 3.8+
- **依赖库**：详见 requirements.txt

## 📦 安装部署

### 1. 环境准备
```bash
# 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 运行软件
```bash
python main.py
```

### 3. 连接硬件（可选）
如需连接物理样机，请：
1. 安装串口驱动（如CH340驱动）
2. 查看可用串口：`COM3`, `COM4` 等
3. 在控制面板中选择串口连接

## 📖 使用指南

### 快速入门
1. 启动软件后，通过顶部Tab切换四大功能模块
2. **参数计算**：输入参数，点击"计算"查看结果
3. **仿真分析**：设置工况参数，运行仿真查看结果
4. **样机控制**：选择连接模式，发送控制指令
5. **知识库**：查看设计方案，保存/加载方案

### 参数化设计
- 高压线半径范围：15-35mm
- 巡检倾角：0-45度
- 机器人重量：15kg（可调）
- 驱动轮数量：4个

### 仿真功能
- 爬坡仿真：支持不同倾角的稳定性分析
- 越障仿真：9阶段越障过程模拟
- 性能指标：速度、扭矩、功率等

### 控制功能
- 支持指令：前进、后退、停止、开合爪、调速等
- 数据记录：自动记录位置、速度、电压、电流等
- 两种模式：虚拟模式（无硬件时）、硬件模式

## 📝 技术架构

### 开发语言与框架
- **Python 3.8+**：主要开发语言
- **PyQt5**：GUI框架
- **NumPy**：数值计算
- **SciPy**：科学计算
- **Matplotlib**：数据可视化
- **PyQtGraph**：高性能实时绘图

### 核心算法
1. **抱紧机构计算**：基于连杆机构几何模型
2. **动力学仿真**：基于牛顿-欧拉方程
3. **稳定性分析**：基于力平衡和摩擦条件
4. **路径规划**：基于越障状态机
