## 📖 系统概述

### 1.1 软件简介
高压线巡检机器人设计与仿真分析平台是一款基于《轮抱式高压线巡检机器人的机构设计与研究》的集成化设计工具。本软件提供参数化设计、运动仿真、样机控制和知识管理四大功能模块，帮助用户完成高压线巡检机器人的设计、分析和验证工作。

### 1.2 适用场景
- 高压线巡检机器人设计工程师
- 科研机构和高校研究人员
- 相关领域学生学习和研究
- 机器人系统集成和测试

### 1.3 主要特点
- 🎯 **参数化设计**：直观的参数输入和结果展示
- 📊 **可视化分析**：丰富的图表和曲线展示
- 🔬 **仿真模拟**：真实的物理模型仿真
- 🎮 **实时控制**：支持虚拟和真实硬件控制
- 💾 **方案管理**：完整的设计方案和实验记录管理

## 🔧 安装指南

### 2.1 系统要求
| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 10 64位 | Windows 11 64位 |
| CPU | Intel i5 | Intel i7 或更高 |
| 内存 | 8GB RAM | 16GB RAM |
| 硬盘 | 2GB可用空间 | 5GB可用空间 |
| 显示器 | 1920×1080 | 2560×1440 |
| Python版本 | 3.8+ | 3.9+ |

### 2.2 安装步骤

#### 2.2.1 Python环境安装
1. 下载Python 3.8+安装包：https://www.python.org/downloads/
2. 运行安装程序，勾选"Add Python to PATH"
3. 安装完成后，打开命令提示符输入：
   ```cmd
   python --version
   ```

#### 2.2.2 安装依赖包
```cmd
# 进入项目目录
cd C:\Users\pc\inspection_robot

# 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2.3 首次启动
```cmd
# 激活虚拟环境后运行
python main.py
```

### 2.4 硬件连接（可选）
如需连接物理样机，请确保：
1. 安装串口驱动（如CH340/CH341驱动）
2. 将样机通过USB转串口线连接到电脑
3. 在控制面板中选择正确的COM端口

## ⚙️ 功能模块详解

### 4.1 参数化设计与计算模块

#### 4.1.1 功能说明
该模块实现论文第3章核心数学模型，包括抱紧机构计算、电机选型和强度校核。

#### 4.1.2 操作流程
1. **参数设置**
   - 高压线半径：15-35mm（推荐25mm）
   - 巡检倾角：0-45度（推荐30度）
   - 机器人重量：10-20kg（默认15kg）

2. **计算类型**
   - 推程计算：显示推程随线径变化曲线
   - 抱紧力计算：显示最小抱紧力要求
   - 电机扭矩：计算不同倾角下的扭矩需求
   - 强度校核：校核连杆应力

3. **结果查看**
   - 数值结果以表格形式展示
   - 曲线以图表形式展示
   - 支持数据导出（CSV格式）

#### 4.1.3 参数说明
| 参数 | 单位 | 范围 | 说明 |
|------|------|------|------|
| R | mm | 15-35 | 高压线半径 |
| β | 度 | 0-45 | 高压线倾角 |
| m | kg | 10-20 | 机器人重量 |
| L1 | mm | - | 连杆1长度 |
| L2 | mm | - | 连杆2长度 |

### 4.2 运动仿真与分析模块

#### 4.2.1 功能说明
基于论文第4章，实现爬坡稳定性和越障过程仿真。

#### 4.2.2 仿真类型
1. **爬坡仿真**
   - 选择倾角：0°, 20°, 40°, 45°
   - 选择线径：15mm, 25mm, 35mm
   - 仿真时长：10秒
   - 输出：位置-时间、速度-时间曲线

2. **越障仿真**
   - 9阶段越障过程模拟
   - 包含：接近、松开、跨越、锁紧等动作
   - 输出：速度、扭矩、相位变化

#### 4.2.3 结果分析
- 仿真曲线自动更新
- 关键指标统计（最大速度、平均速度等）
- 支持多组结果对比

### 4.3 样机测试与控制模块

#### 4.3.1 连接模式
- **虚拟模式**：无需硬件，软件模拟运行
- **硬件模式**：通过串口连接真实样机

#### 4.3.2 控制指令
| 指令 | 功能 | 说明 |
|------|------|------|
| forward | 前进 | 向前移动 |
| backward | 后退 | 向后移动 |
| stop | 停止 | 停止所有动作 |
| grip_open | 开爪 | 打开夹爪 |
| grip_close | 闭爪 | 关闭夹爪 |
| speed_up | 加速 | 提高移动速度 |
| speed_down | 减速 | 降低移动速度 |

#### 4.3.3 数据记录
- 自动记录时间戳、位置、速度、电压、电流
- 数据导出为JSON格式
- 支持实验数据分析

### 4.4 知识库与方案管理模块

#### 4.4.1 知识库内容
- **设计指标**：论文表2-1的10项设计指标
- **研究综述**：国内外6个高压线巡检机器人案例
- **技术参数**：关键部件性能参数

#### 4.4.2 方案管理
- **保存方案**：将当前参数保存为JSON文件
- **加载方案**：读取之前保存的方案
- **管理实验**：查看历史实验记录

## ❓ 常见问题

### 5.1 安装问题

**Q：Python环境安装失败**
```
解决方案：
1. 关杀毒软件
2. 以管理员身份运行安装程序
3. 确保磁盘空间充足
```

**Q：pip安装依赖包失败**
```
解决方案：
1. 更新pip：python -m pip install --upgrade pip
2. 使用国内镜像：
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 5.2 运行问题

**Q：软件启动后界面显示异常**
```
解决方案：
1. 检查显示器分辨率
2. 更新显卡驱动
3. 重启软件
```

**Q：串口连接失败**
```
解决方案：
1. 检查串口线连接
2. 安装串口驱动
3. 更换COM端口
4. 检查设备管理器中的COM口
```

### 5.3 功能问题

**Q：计算结果与论文不符**
```
原因：
1. 参数设置错误
2. 版本差异

解决方案：
1. 检查参数范围
2. 参考MOD_DEF.md中的公式修正
```

**Q：仿真曲线不显示**
```
解决方案：
1. 确保matplotlib安装正确
2. 检查numpy版本
3. 重启软件
```
